Cross-Platform Compatibility:
  • ✅ Windows: Consolas, Lucida Console, Courier New
  • ✅ Mac: Consolas, Monaco, Menlo, Courier New
  • ✅ Android/iOS: Courier New, monospace (generic)
  • ❌ Monaco/Menlo: NOT available on Windows or Android - will fall back to next font in stack
Recommended Font Stack for Maximum Compatibility:
"Consolas", "Lucida Console", "Monaco", "Menlo", "Courier New", monospace

This stack ensures:

  • Windows users get Consolas or Lucida Console
  • Mac users get Consolas, Monaco, or Menlo
  • Mobile/other platforms fall back to Courier New or generic monospace
  • All fonts maintain uniform spacing for proper alignment
