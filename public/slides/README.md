# Campaign Slides

This folder contains the campaign slides that are displayed in the slide viewer.

## Usage

1. Generate slides from the Admin panel (Admin > Generate Campaign Slides)
2. The generated slides will be saved as a ZIP file
3. Extract the ZIP file and copy the slides to this folder
4. Slides should be named sequentially: `slide_1.jpg`, `slide_2.jpg`, `slide_3.jpg`, etc.

## Viewing Slides

There are two ways to view the slides:

1. **Authenticated Users**: Click the "Campaigns" button in the bottom navigation (requires sign-in)
2. **Public Access**: Share the URL `https://yourdomain.com/view-slides` - anyone can view without signing in

## File Naming Convention

- Files must be named: `slide_1.jpg`, `slide_2.jpg`, `slide_3.jpg`, etc.
- The slide viewer automatically detects all slides starting from `slide_1.jpg`
- Slides must be in JPG format
